# Spotify Clone – Full Stack Music Streaming Platform

A full-stack Spotify-inspired music streaming application that allows users to browse albums, stream songs, and enjoy a modern music player experience. The platform also includes a dedicated admin dashboard for managing songs, albums, and media assets.

Built using React, Node.js, Express, MongoDB, and Cloudinary, the application demonstrates real-world full-stack development concepts, including media management, REST APIs, cloud storage integration, and responsive UI design.

---

## 🚀 Project Overview

Spotify Clone is a full-stack music streaming platform designed to replicate the core functionality of modern music applications. Users can explore albums, browse songs, and stream music through an intuitive interface, while administrators can manage content through a dedicated dashboard.

The application uses a React frontend, Express backend, MongoDB database, and Cloudinary cloud storage for handling audio and image uploads.
